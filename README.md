# Elzo Brito - Consultoria em IA

Site profissional estático para GitHub Pages, criado para divulgar consultorias, capacitações e projetos em Inteligência Artificial aplicada à Educação, Automação e Governança.

## Estrutura

```text
/
├── index.html
├── style.css
├── script.js
├── README.md
└── assets/
    └── .gitkeep
```

## Como testar localmente

Abra o arquivo `index.html` diretamente no navegador. O site não usa frameworks, dependências externas nem processo de build.

## Como editar o conteúdo

- Textos e seções principais: edite `index.html`.
- Cores, espaçamentos, responsividade e temas: edite `style.css`.
- Alternância de tema, menu mobile e ano do rodapé: edite `script.js`.
- Foto profissional opcional: adicione uma imagem em `assets/foto-elzo.jpg`.

## Contatos e placeholders

Os contatos reais já incluídos são:

- E-mail: `elzobrito@gmail.com`
- GitHub: `https://github.com/elzobrito`
- LinkedIn: `https://www.linkedin.com/in/elzobrito`

O botão de contato adicional usa o mesmo e-mail real.

## Publicação no GitHub Pages

1. Envie estes arquivos para o repositório `elzobrito.github.io`.
2. No GitHub, acesse `Settings`.
3. Entre em `Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha a branch principal, normalmente `main`, e a pasta `/root`.
6. Salve a configuração.

Após a publicação, o site ficará disponível em:

```text
https://elzobrito.github.io
```

## Observações

- O site é compatível com GitHub Pages.
- A página usa `lang="pt-BR"`, SEO básico e Open Graph.
- O layout funciona sem foto profissional.
- Os canais de contato usam apenas e-mail, GitHub e LinkedIn real.
