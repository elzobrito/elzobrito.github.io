## Core idea

The original post makes the central ESAA argument: agents should emit intentions into a governed workflow rather than mutate code directly without an auditable trail.

Event sourcing gives the process memory. The code changes matter, but so do claims, evidence, review, and the sequence of decisions that made the change legitimate.
